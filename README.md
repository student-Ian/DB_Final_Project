## 關於本專案

本組的期末專案主題為線上外送平台Delish Way，旨在提供顧客一次看見所有餐廳而不需要花時間思考的點餐體驗。Delish Way分成顧客和老闆兩種角色，分別可以進行點餐和處理訂單，同時支援優惠券使用、歷史訂單查詢和評分等功能。
